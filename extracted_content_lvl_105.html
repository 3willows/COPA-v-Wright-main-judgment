The arrows.meta library

553.

TikZ is a large package that is used to create graphics in LaTeX. It allows pictures to be defined programmatically and, given its complexity, is broken down into many different libraries with additional functionalities and features {Rosendahl1 [143] {G/7/47}}.

554.

The White Paper LaTeX Files make use of the arrows.meta library in TikZ {L21/9.1/3}. That library was only released in September 2013 {Rosendahl1 [145] {G/7/48}}. Any file that loaded the arrows.meta library could not have been created in 2009 {Rosendahl1 [146] {G/7/48}}.
