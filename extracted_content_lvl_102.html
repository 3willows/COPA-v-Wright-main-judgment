hidelinks

545.

The “hyperref” package in LaTeX defines commands to add hyperlinks to a PDF file compiled in LaTeX. Dr Wright’s White Paper LaTeX Files purport to call on a “hidelinks” option from that package {e.g. at {L21/9.1/4}. That option hides the fact that links within the document are hyperlinks, by displaying them without underlining {Rosendahl1 [130] {G/7/45}}. Mr Rosendahl explained that the “hidelinks” option was only added to the hyperref package in 2010 (i.e. after the Bitcoin White Paper) {Rosendahl1 [130] {G/7/45}}.
